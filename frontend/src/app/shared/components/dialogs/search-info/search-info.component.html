help  Ayuda para búsquedas

  • Las búsquedas son insensibles a las mayúsculas y minúsculas para todos los campos tipo texto.

  • Se admite el uso de expresiones regulares para todos los campos de búsqueda de tipo texto.

Metacaracteres básicos para la búsqueda con expresiones regulares

. Cualquier carácter excepto salto de linea.
A.A  arrow_forward  ANA

* Indica que el carácter precedente puede ocurrir 0 o más veces.
A.*A  arrow_forward  AA | ANA | ANNA

+ Indica que el carácter precedente puede ocurrir 1 o más veces.
A.+A  arrow_forward  ANA

? Indica que el carácter precedente puede ocurrir 0 o 1 vez.
A.?A  arrow_forward  AA | ANA
