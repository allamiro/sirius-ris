  • Searches are case-insensitive for every text field.

Basic metacharacters for regular expression searches

. Any character except a line break.
A.A  arrow_forward  ANA

* Indicates that the preceding character can appear zero or more times.
A.*A  arrow_forward  AA | ANA | ANNA

+ Indicates that the preceding character can appear one or more times.
A.+A  arrow_forward  ANA

? Indicates that the preceding character can appear zero or one time.
A.?A  arrow_forward  AA | ANA
